# Lista de Exercícios - Programação II
A lista de exercícios tem como objetivo praticar conceitos fundamentais da linguagem JavaScript, incluindo:

- Funções
- Arrays
- Strings
- Objetos Literais
- Classes
- Métodos
- Estruturas de Repetição
- Arrow Functions
- Manipulação de Dados

## Exercícios Desenvolvidos

### 1. O Detetive de Divisores
Implementação de duas soluções para contagem de divisores:
- Método de força bruta
- Método otimizado utilizando a raiz quadrada do número (Trial Division Algorithm)

### 2. Inversão de Arrays
Inversão manual de um vetor sem utilizar o método `.reverse()`.

### 3. Filtragem de Números Pares
Percorrendo arrays e identificando números pares utilizando laços de repetição.

### 4. Correção de Nomes
Manipulação de strings utilizando:
- `.toUpperCase()`
- `.toLowerCase()`

### 5. Extração de Domínios de E-mail
Utilização dos métodos:
- `.indexOf()`
- `.slice()`

### 6. Verificação de Acesso
Comparação de senhas ignorando:
- Espaços em branco
- Diferença entre letras maiúsculas e minúsculas

### 7. Saudações em Múltiplos Idiomas
Uso de parâmetros padrão em funções.

### 8. Conversão de Temperaturas
Utilização de Arrow Functions.

### 9. Percorrendo Arrays com forEach
Exibição de elementos de um array utilizando `.forEach()`.

### 10. Inventário de Produtos
Criação de objetos literais com propriedades e métodos.

### 11. Relatório de Usuários
Percorrendo vetores de objetos e filtrando usuários ativos.

### 12. Média da Turma
Cálculo de média aritmética utilizando arrays de objetos.

### 13. Fábrica de Carros
Introdução ao conceito de Classes em JavaScript:
- Constructor
- Métodos
- Instanciação com `new`

### 14. Simulador de Conta Bancária
Manipulação de estado através de classes:
- Depósito
- Saque
- Controle de saldo
